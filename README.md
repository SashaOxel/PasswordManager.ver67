# Password Manager

Консольный менеджер паролей с шифрованием хранилища и генератором паролей.

## Требования

- **Windows** (приложение использует WinAPI для настройки кодировки консоли)
- **g++** с поддержкой C++17 — рекомендуется [MinGW-w64](https://www.mingw-w64.org/) или [MSYS2](https://www.msys2.org/)

### Установка MinGW через MSYS2 (рекомендуется)

1. Скачайте и установите [MSYS2](https://www.msys2.org/)
2. Откройте **MSYS2 UCRT64** и выполните:
   ```
   pacman -S mingw-w64-ucrt-x86_64-gcc
   ```
3. Добавьте `C:\msys64\ucrt64\bin` в переменную среды `PATH`

### Проверка установки

```
g++ --version
```

## Сборка

Перейдите в папку проекта и запустите скрипт сборки:

```
cd PasswordManager.ver67
build.bat
```

После успешной сборки появится файл `PasswordManager.exe`.

### Ручная сборка (альтернатива)

```
g++ -std=c++17 -O2 main.cpp src/crypto.cpp src/generator.cpp src/menu.cpp src/records.cpp -I include -o PasswordManager.exe
```

## Запуск

```
PasswordManager.exe
```

## Функциональность

### Хранилище паролей
- Создание нового хранилища с мастер-паролем
- Вход в существующее хранилище
- Хранилище сохраняется в файл `vault_<имя>.dat` рядом с исполняемым файлом

### Управление записями
- Просмотр всех записей (название, логин, пароль, URL, заметка)
- Добавление новой записи
- Редактирование существующей записи
- Удаление записи
- Вывод пароля конкретной записи

### Генератор паролей
- Быстрая генерация (12 символов, буквы + цифры)
- Настраиваемая генерация (длина 4–64, выбор типов символов)
- Генерация нескольких паролей сразу (до 20)

### Шифрование файлов
- Зашифровать произвольный файл (алгоритм: бегущий ключ XOR)
- Расшифровать файл

## Структура проекта

```
PasswordManager.ver67/
├── main.cpp              # Точка входа
├── build.bat             # Скрипт сборки
├── include/
│   ├── crypto.h          # Шифрование
│   ├── generator.h       # Генератор паролей
│   ├── menu.h            # Меню
│   └── records.h         # Записи хранилища
└── src/
    ├── crypto.cpp
    ├── generator.cpp
    ├── menu.cpp
    └── records.cpp
```

## Файлы хранилища

Хранилища создаются в той же папке, откуда запущен `PasswordManager.exe`, с именами вида `vault_<имя>.dat`. Файлы зашифрованы мастер-паролем — без него содержимое прочитать невозможно.
